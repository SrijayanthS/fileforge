FileForge

A browser-based image and PDF toolkit for resizing, converting, compressing, and managing files entirely in your browser.

Live Demo: https://srijayanths.github.io/fileforge/

Features

 Image Tools
- Resize images
- Convert JPG, PNG, and WebP
- Compress images
- Maintain aspect ratio
- Preview and download

 PDF Tools
- Merge PDFs
- Split PDFs
- Convert images to PDF
- Display PDF metadata

 Highlights
- Responsive design
- Light & Dark mode
- Drag-and-drop uploads
- Local browser processing (no server uploads)

 Tech Stack
- HTML5
- CSS3
- JavaScript
- Canvas API
- PDF-lib

 Run Locally

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Privacy

All files are processed locally in your browser and are never uploaded to a server.
